# NexaResume - AI Resume Builder 🚀
**NexaResume** is a high-performance, full-stack AI Resume Builder designed to help users create, optimize, and manage professional resumes with ease. By leveraging the MERN stack and Google Gemini AI, it provides intelligent content optimization and seamless resume management.

---

## 🌟 Key Features
1. **User Authentication:** Secure Sign-In and Sign-Up functionality.
2. **AI-Powered Optimization:** Integrated **Google Gemini AI** to improve and refine resume content.
3. **Live Preview & Sharing:** Real-time resume previewing with a shareable online link.
4. **Media Management:** Upload profile images with integrated background removal via **ImageKit**.
5. **Resume Management:** Full CRUD operations (Add, Edit, Delete) for multiple resume profiles.
6. **Modern Templates:** Choose from various professional resume templates.
7. **Deployment Ready:** Configured for deployment on Vercel and modern cloud environments.
